🔧 Firebase 설정 가이드
총 소요 시간: 약 10~15분 (개발 경험 불필요)

① Firebase 프로젝트 만들기

  1. console.firebase.google.com 접속 → Google 계정 로그인
  2. 「프로젝트 추가」 클릭 → 이름 입력 (예: jakae-community) → 계속
  3. Google 애널리틱스 비활성화 → 「프로젝트 만들기」

② 웹 앱 등록 및 설정 키 복사

  1. 프로젝트 홈 → 「</> 웹 앱 추가」 클릭
  2. 앱 닉네임 입력 → 「앱 등록」
  3. 화면에 표시되는 firebaseConfig 내용 전체 복사
  4. 이 HTML 파일 상단 firebaseConfig 부분에 붙여넣기

③ Firestore 데이터베이스 활성화

  1. 왼쪽 메뉴 「Firestore Database」 → 「데이터베이스 만들기」
  2. 「테스트 모드에서 시작」 선택 → 위치 asia-northeast3 (Seoul) → 「완료」

④ Storage 활성화 (이미지 업로드용)

  1. 왼쪽 메뉴 「Storage」 → 「시작하기」
  2. 「테스트 모드에서 시작」 → 위치 확인 → 「완료」

⑤ 완료!

설정 후 HTML 파일을 저장하고 브라우저를 새로고침하면 상단에 ✅ Firebase 연결됨 표시가 나타납니다.

💡 무료 한도: Firebase 무료 플랜(Spark)으로 하루 5만 읽기 / 2만 쓰기 가능 → 30명 소규모 커뮤니티엔 충분합니다!
